☺ Perguntas para prova ☺

1- O que faz um UI designer?

Resposta: Como todo designer gráfico, o objetivo de um UI Designer é resolver problemas visuais. Ele simplifica a vida das pessoas ao proporcionar experiências satisfatórias na realização de tarefas do cotidiano, principalmente no ambiente digital. Para isso, este profissional fica responsável pela construção de interfaces. Sua função é dar aos sites, apps, redes sociais, programas a cara que eles têm para o público.

2- O que faz um UX designer?

Resposta: Os UX designers têm como objetivo melhorar a experiência de um usuário durante a atualização de um produto, serviço ou sistema. Isso é feito implementando um processo de design thinking que, de acordo com a d.school da Universidade de Stanford, é dividido em cinco etapas: empatizar, definir, idealizar, prototipar e testar.

3- Como anda o mercado brasileiro para UI designers?

Resposta: O mercado de UI Design está em grande expansão. A oferta de cursos e treinamentos para a área de UX/UI Design é evidente, o que é bom, visto que existe de fato uma necessidade enorme desse tipo de profissional no mercado brasileiro e mundial. Muitas empresas já têm nascido com produtos digitais em sua essência, enquanto outras com produtos já existentes precisam estar em constante evolução e consolidação a seu mercado; e para que atinjam seus objetivos de negócios e as necessidades de seus usuários, profissionais capacitados e preparados tem sido muito procurados para suprir essa demanda que não pára de crescer, para atuar nos mais diferentes níveis, indústrias e projetos.

4- Onde pode trabalhar um UI designer?

Resposta: O UI designer pode trabalhar em equipes de Design, departamentos de TI de grandes empresas e seguir uma carreira de designer freelancer

5- Quanto ganha um UI designer e um UX designer?

Resposta: Um UI/UX designer pode ganhar entre R$ 3.000 e R$ 12.000

6- Existe algum requisito para começar a aprender sobre UI design?

Resposta: Não, não existe, qualquer pessoa pode ingressar nessa área, mas quem já tem algum tipo de experiência na área de design fica mais fácil.

7- Onde encontramos um UI designer?

Resposta: Podemos encontrar um UI designer em agências de design e desenvolvimento.

8- Qual a diferença entre UI e UX designer?

Resposta: O UI designer tem como objetivo principal cuidar da interface de algum aplicativo ou site, já o UX designer tem como objetivo agradar os usuários na utilização desse aplicativo ou site, e que o use mais vezes.

9- Web design e UI design não são parecidos?

Resposta: Eles tem suas semelhanças mas cada um faz o seu papel, dependendo da situção do aplicativo ou site. E por serem do meio de design, eles são semelhantes.

10- Qual a importância da prática em UI/UX?

Resposta: Adquirir conhecimento e crescer no mercado de design que está crescendo cada vez mais.

11- O que significa design?

Resposta: Design significa criar maneiras para solucionar problemas ou criar qualquer coisa em algum meio de técnico ou estético.

12- Quando um processo de UX design é implementado?

Resposta: Com a definição do projeto compreensão do usuário.

13- O que é user flow (fluxo de usuário)?

Resposta: User flow é uma técnica que te permite mapear todo fluxo de telas do seu site ou aplicativo de forma rápida.

14- Como implementar UX no dia-a-dia?

Resposta: Tirar a melhor experiência de alguma situação.

15- Como Implementar UI no dia-a-dia?

Resposta: Com organização e clareza
